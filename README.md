## ğŸš€ Panduan Setup

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di mesin lokal Anda:

### 1. Ambil Pembaruan Terakhir
Pastikan repositori Anda sinkron dengan versi terbaru:
```bash
git pull origin main
```

### 2. Clone Repositori (Jika belum)
```bash
git clone https://github.com/af133/kopma.git
cd kopma
```

### 3. Instalasi Dependensi
Jalankan perintah berikut untuk mengunduh semua package Flutter yang diperlukan:
```bash
flutter pub get
```

### 4. Jalankan Aplikasi
Pastikan emulator atau perangkat fisik sudah terhubung, lalu jalankan:
```bash
flutter run
```
Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
